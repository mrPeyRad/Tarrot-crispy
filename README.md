# Telegram-бот: таро, астрология и дневник предсказаний

Локальный MVP телеграм-бота, которого можно запускать без тяжёлых фреймворков. Он уже работает в личных чатах и группах, хранит профиль пользователя в SQLite, умеет вести дневник предсказаний и присылать рассылки по расписанию.

## Что умеет сейчас

- `карта дня` или `/card` — случайная карта таро с картинкой и трактовкой.
- `/spread3` — расклад на 3 карты: прошлое, настоящее, будущее.
- `/yesno [вопрос]` — быстрый ответ в формате «Да / Скорее да / Пока неясно / Скорее нет / Нет».
- `/relationship` — карта отношений.
- `/cardinfo [название карты]` — мини-энциклопедия карты с прямым и перевёрнутым значением.
- `/deck [название]` — переключить визуал колоды.
- `/horoscope [знак]` — гороскоп на день.
- `/week [знак]` — гороскоп на неделю.
- `/moon` — лунный календарь с фазой луны и советом на день.
- `/compat [знак]` или `/compat [знак] [знак]` — совместимость знаков зодиака.
- `/astroalert` — астросигнал дня.
- `/rune` — руна дня.
- `/8ball [вопрос]` — шар предсказаний.
- `/journal` — дневник предсказаний и краткая статистика по текущему месяцу.
- `/subscribe [daily|weekly]` — включить рассылку.
- `/unsubscribe` — выключить рассылку.
- `/setsign [знак]` — сохранить знак зодиака в профиль.
- `/profile` — показать профиль и последние расклады.
- `/cancel` — сбросить текущий диалог.

Текстовые триггеры тоже работают: `карта дня`, `гороскоп`, `гороскоп на неделю`, `лунный календарь`, `совместимость`, `астроалерт`, `руна дня`, `расклад 3 карты`, `да/нет`, `карта отношений`, `значение карты`, `дневник предсказаний`, `рассылка`.

## Визуалы колод

Сейчас доступны 3 визуальных режима:

- `Классическая колода Райдера-Уэйта` — реальные классические изображения карт.
- `Стилизованный визуал Тота` — тёмная мистическая тема.
- `Минималистичная колода` — лаконичный светлый стиль.

Важно: только Rider-Waite использует классические изображения карт. Остальные режимы сейчас сделаны как стилизованные визуальные темы, чтобы уже можно было переключать атмосферу бота без сложной графической подготовки.

## Дневник предсказаний

Бот сохраняет в журнал:

- дневные и недельные гороскопы;
- карту дня и карту недели;
- расклады таро;
- ответы `Да/Нет`;
- руны дня;
- ответы шара предсказаний;
- лунный календарь;
- астро-алерты.

Команда `/journal` показывает:

- краткую статистику за текущий месяц;
- последние записи;
- пометку, если запись пришла по автоматической рассылке.

## Рассылки

Поддерживаются 2 режима:

- `daily` — каждый день в `09:00` по локальному времени сервера.
- `weekly` — раз в неделю после `09:00` по локальному времени сервера.

Что приходит в рассылке:

- ежедневная — гороскоп на день + карта дня;
- еженедельная — гороскоп на неделю + карта недели.

Важно: рассылка работает только пока процесс бота запущен. Если бот остановлен, сообщения в этот период отправляться не будут.

## Структура проекта

- `main.py` — точка входа.
- `app/bot.py` — polling-бот, сценарии общения и рассылки.
- `app/database.py` — SQLite-хранилище профиля, истории, дневника и подписок.
- `app/tarot.py` — колода из 78 карт, расклады, поиск карт, значения и визуалы колод.
- `app/horoscope.py` — знаки зодиака, гороскоп на день и неделю.
- `app/cosmic.py` — лунный календарь, совместимость знаков и астро-сигналы.
- `app/mystic.py` — руна дня и шар предсказаний.
- `app/telegram_api.py` — тонкая обёртка над Telegram Bot API.

## Как запустить

1. Создай бота в `@BotFather` и получи токен.
2. Скопируй `.env.example` в `.env`.
3. Заполни `BOT_TOKEN`.
4. При желании измени `DATABASE_PATH`, если хочешь хранить SQLite-файл в другом месте.
5. Запусти:

```bash
python main.py
```

## Переменные окружения

```env
BOT_TOKEN=твой_токен
DATABASE_PATH=bot_data.sqlite3
POLLING_TIMEOUT=30
REQUEST_TIMEOUT=40
```

## Примеры использования

- `/setsign Овен` — сохранить знак сразу одной командой.
- `/horoscope` — если знак уже сохранён, бот ответит сразу.
- `/week` — если знак сохранён, бот пришлёт недельный прогноз.
- `/deck Тота` — переключить визуал карт.
- `/compat Лев` — сравнить сохранённый знак с Львом.
- `/compat Овен Весы` — сравнить два знака без сохранения в профиль.
- `/journal` — открыть дневник и посмотреть статистику.
- `/subscribe daily` — включить ежедневную рассылку.
- `/subscribe weekly` — включить еженедельную рассылку.
- `/8ball Стоит ли писать первым?` — получить ответ от шара предсказаний.
- `/cardinfo Луна` — показать карточку-энциклопедию.

## Групповые чаты

Если хочешь, чтобы бот реагировал на обычные фразы вроде `карта дня`, `гороскоп` или `руна дня` в группе, отключи privacy mode в `@BotFather`:

1. `/setprivacy`
2. выбрать бота
3. `Disable`

Если privacy mode включён, бот обычно видит только команды, ответы на свои сообщения и явные упоминания.

# Telegram-бот: таро, расклады и гороскоп

Локальный MVP телеграм-бота, которого можно запускать без внешних Python-зависимостей. Он уже умеет работать и в личных чатах, и в группах.

## Что умеет сейчас

- `карта дня` или `/card` — случайная карта таро с картинкой и трактовкой.
- `/spread3` — расклад на 3 карты: прошлое, настоящее, будущее.
- `/yesno [вопрос]` — быстрый ответ в формате «Да / Скорее да / Пока неясно / Скорее нет / Нет».
- `/relationship` — карта отношений.
- `/cardinfo [название карты]` — мини-энциклопедия: прямое и перевёрнутое значение карты.
- `/horoscope [знак]` — гороскоп на день.
- `/setsign [знак]` — сохранить знак зодиака в профиль.
- `/profile` — показать профиль и недавние расклады.
- `/cancel` — сбросить текущий диалог.

Текстовые триггеры тоже работают: `карта дня`, `гороскоп`, `расклад 3 карты`, `да/нет`, `карта отношений`, `значение карты`.

## Структура проекта

- `main.py` — точка входа.
- `app/bot.py` — polling-бот и сценарии общения.
- `app/database.py` — SQLite-хранилище профиля, истории и состояния диалогов.
- `app/tarot.py` — колода из 78 карт, расклады, поиск карт и значения.
- `app/horoscope.py` — знаки зодиака и генерация гороскопа.
- `app/telegram_api.py` — тонкая обёртка над Telegram Bot API.

## Как запустить

1. Создай бота в `@BotFather` и получи токен.
2. Скопируй `.env.example` в `.env`.
3. Заполни `BOT_TOKEN`.
4. При желании измени `DATABASE_PATH`, если хочешь хранить SQLite-файл в другом месте.
5. Запусти:

```bash
python main.py
```

## Переменные окружения

```env
BOT_TOKEN=твой_токен
DATABASE_PATH=bot_data.sqlite3
POLLING_TIMEOUT=30
REQUEST_TIMEOUT=40
```

## Профиль и память бота

Бот теперь хранит данные в SQLite:

- знак зодиака пользователя;
- текущую выбранную колоду;
- состояние активного диалога;
- историю выпавших карт и раскладов.

По умолчанию база создаётся как `bot_data.sqlite3` рядом с проектом. Этот файл уже добавлен в `.gitignore`.

## Использование

- `/start` или `/help` — список возможностей.
- `/setsign Овен` — сохранить знак сразу одной командой.
- `/horoscope` — если знак уже сохранён, бот ответит сразу.
- `/cardinfo Луна` — показать карточку-энциклопедию.
- `/yesno Стоит ли писать первым?` — вытянуть карту для быстрого ответа.

## Групповые чаты

Если хочешь, чтобы бот реагировал на обычные фразы вроде `карта дня` и `гороскоп` в группе, отключи privacy mode в `@BotFather`:

1. `/setprivacy`
2. выбрать бота
3. `Disable`

Если privacy mode включён, бот обычно видит только команды, ответы на свои сообщения и явные упоминания.
